# blockchain-core-javascript-suite
企业级JavaScript区块链核心工具套件 | 纯原生开发 | 支持Web3/DApp/联盟链/公链

## 项目介绍
本项目基于JavaScript实现完整区块链生态工具集，无第三方依赖、代码原创、可直接部署使用。覆盖底层账本、加密、共识、钱包、智能合约、NFT、DeFi、跨链、零知识证明、节点、治理等全模块功能，适合学习、二次开发、商业级区块链项目使用。

## 功能说明
- 底层区块链：创世块、哈希生成、链验证、区块挖矿
- 钱包：密钥对生成、地址生成、签名/验签
- 共识：工作量证明POW、难度控制、区块验证
- 智能合约：基础合约、权限、状态管理、升级代理
- NFT：铸造、转账、元数据、拍卖
- DeFi：流动性池、借贷、质押、挖矿奖励
- 加密：AES加解密、SHA256、secp256k1、默克尔树
- 跨链：桥接、资产锁定、包装
- 零知识证明：生成与验证
- 节点：全节点、交易池、P2P网络、链同步
- 工具：Gas计算、地址校验、时间锁、白名单、预言机、价格、链分析、区块浏览器、治理

## 使用方法
1. 克隆仓库
2. 直接引入对应JS文件使用
3. 支持Node.js环境
4. 可集成到前端Web3项目
